# YouTube Shorts Blocker

YouTube Shortsを非表示にして、通常の動画に集中できるようにするChrome拡張機能です。

## 特徴

- YouTube Shortsのセクションを完全に非表示
- Shorts動画ページへのアクセスをブロック
- シンプルで軽量、余計な機能なし
- 個人情報の収集なし、完全にローカルで動作

## インストール方法

### Chrome Web Storeから（審査中）

現在Chrome Web Storeで審査中です。公開されたらリンクを追加します。

### 手動インストール（開発者向け）

1. このリポジトリをダウンロード（「Code」→「Download ZIP」）
2. Chromeで `chrome://extensions/` を開く
3. 右上の「デベロッパーモード」をON
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. ダウンロードしたフォルダを選択

## 使い方

インストールするだけで自動的に有効になります。設定不要です。
